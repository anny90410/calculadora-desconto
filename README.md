#calculadora-desconto
## Desconto Rápido
Este produto serve para calcular a contidade de produtos e o valor final. Para que ele funciona, usamos HTML, CSS, JavaScript.

## Como rodar o projeto
1. Baixe o repositório(clique no Botão"Code"> Download ZIP)
2. Extraia a pasta
3. Abra o arquivo index.html no navegador
